# bill.y2k Command Dashboard v3

Rebrand from VGL to bill.y2k with black/metal/green resale dashboard theme.

## New
- bill.y2k app name and icon
- black/metal/green UI
- Vinted tab
- 21 preloaded clothing items
- item status: Not listed / Listed / Sold
- sold price tracking
- sold total and sold profit update automatically
- potential profit and listed value tracking
- existing project, habit grid, tasks, lead/content vault remain

## Replace in GitHub
Upload these files into the existing vgl-dashboard repo:
- index.html
- styles.css
- app.js
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png
- README.txt

Then commit changes.

If the home screen app shows the old version:
1. Open the GitHub Pages link in Safari
2. Refresh
3. Close the home screen app
4. Reopen
